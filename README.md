To run this project locally,

```bash
mkdir bookish
cd bookish
git clone https://github.com/Aritra8438/Demo_E-commerce_Site.git
virtualenv bookish
cd frontend
npm install
npm run serve
cd ../backend
python manage.py runserver
```
---
